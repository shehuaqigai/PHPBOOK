1.yum install gdb安装gdb

2.先修改config.m4

在其中加入：
if test -z "$PHP_DEBUG" ; then AC_ARG_ENABLE(debug, [--enable-debug compile with debugging system], [PHP_DEBUG=$enableval],[PHP_DEBUG=no] ) fi

3.在扩展目录执行/opt/lampp/bin/phpize

4./configure的时候加入 –enable-debug

./configure --with-php-config=/opt/lampp/bin/php-config --enable-debug

4.make

5.make install

6.配置php.ini,重启apache

7.在扩展目录执行gdb /opt/lampp/bin/php(指定php的解释器)进入gdb操作空间
